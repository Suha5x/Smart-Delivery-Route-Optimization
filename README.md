# Smart Delivery Route Optimization System

This project implements a Smart Delivery Route Optimization System using Dijkstra and Greedy Algorithms.

## Features

* Shortest path calculation using Dijkstra Algorithm
* Delivery traversal using Greedy Algorithm
* Interactive ReactJS dashboard
* Live route visualization
* Delivery distance tracking

## Technologies Used

* ReactJS
* JavaScript
* Graph Algorithms
* Dijkstra Algorithm
* Greedy Algorithm

## Algorithms Used

### Dijkstra Algorithm

Used for finding shortest paths between delivery locations.

### Greedy Algorithm

Used for generating optimized delivery traversal routes.
